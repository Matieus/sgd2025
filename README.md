# SGD 

## Setup
-----------------------------------------------------------------
#### Create virtual environment for python
Execute the below command to create virtual environment
- ``` python -m venv venv ```


for windows:
- ```source venv/Scrips/activate```

for unix:
- ```source venv/bin/activate```
-----------------------------------------------------------------
#### Install pygame
- ```pip install pygame```
